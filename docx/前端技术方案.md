# 二、设计系统

## 2.1 色彩体系（OKLCH）

```css
:root {
  --primary:       oklch(70% 0.18 45);   /* 温暖橙 - 主色调 */
  --primary-light: oklch(85% 0.1 45);    /* 浅橙 - 背景高亮 */
  --bg:            oklch(98% 0.005 80);  /* 暖白 - 页面背景 */
  --card:          oklch(99% 0.005 80);  /* 卡片白 */
  --text:          oklch(25% 0.02 50);   /* 深暖灰 - 主文字 */
  --text-secondary:oklch(55% 0.02 50);   /* 中灰 - 辅助文字 */
  --success:       oklch(72% 0.15 160);  /* 成功绿 - 解锁状态 */
  --border:        oklch(90% 0.01 80);   /* 边框色 */
}
```

**TailwindCSS 映射**

| 变量名 | Tailwind Class | 色值 |
|--------|---------------|------|
| primary | `bg-primary` `text-primary` | oklch(70% 0.18 45) |
| bg | `bg-bg` | oklch(98% 0.005 80) |
| card | `bg-card` | oklch(99% 0.005 80) |
| text | `text-text` | oklch(25% 0.02 50) |
| success | `text-success` | oklch(72% 0.15 160) |
| border | `border-border` | oklch(90% 0.01 80) |

## 2.2 风格：Claymorphism

**核心特征**
- 圆角：16-24px
- 边框：3-4px solid
- 阴影：inset + outer 双层阴影（软 3D 效果）
- 交互：按下时 inset 阴影 + scale(0.97)

**CSS 实现**

```css
/* 卡片样式 */
.clay-card {
  background: var(--card);
  border: 3px solid var(--border);
  border-radius: 20px;
  box-shadow: 
    6px 6px 12px oklch(85% 0.01 80),
    -2px -2px 8px oklch(100% 0 0),
    inset 0 1px 2px oklch(100% 0 0);
}

/* 按钮样式 */
.clay-btn {
  background: var(--primary);
  border: 3px solid oklch(65% 0.18 45);
  border-radius: 16px;
  box-shadow: 
    4px 4px 8px oklch(85% 0.01 80),
    -2px -2px 6px oklch(100% 0 0),
    inset 0 1px 2px oklch(100% 0 0);
}

.clay-btn:active {
  box-shadow: 
    inset 2px 2px 4px oklch(60% 0.18 45),
    inset -1px -1px 3px oklch(80% 0.1 45);
  transform: scale(0.97);
}
```

# 五、API 对接

## 5.1 接口封装（src/api.js）

```javascript
const API_BASE = '/api'

async function request(url, options = {}) {
  const res = await fetch(`${API_BASE}${url}`, options)
  if (!res.ok) {
    const err = await res.json().catch(() => ({ detail: '请求失败' }))
    throw new Error(err.detail || `HTTP ${res.status}`)
  }
  return res.json()
}

export function getCats() {
  return request('/cats')
}

export function recognize(file) {
  const form = new FormData()
  form.append('file', file)
  return request('/recognize', { method: 'POST', body: form })
}

export function createSighting({ catId, location, confidence, file }) {
  const form = new FormData()
  form.append('cat_id', catId)
  if (location) form.append('location', location)
  if (confidence !== undefined) form.append('confidence', confidence)
  if (file) form.append('file', file)
  return request('/sightings', { method: 'POST', body: form })
}

export function getUserProfile() {
  return request('/user/profile')
}
```

# 六、启动方式

## 6.1 开发环境

```bash
cd cat-frontend
npm install
npm run dev
```

访问：http://localhost:5173

## 6.2 生产构建

```bash
npm run build
# 输出到 dist/ 目录
```

## 6.3 注意事项

- 后端服务需运行在 `localhost:8000`
- 前端通过 Vite 代理转发 API 请求到后端
- 地图使用 OpenStreetMap 瓦片，无需 API Key
