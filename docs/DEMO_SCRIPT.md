# 演示脚本

面向评审 / 新成员的 5-8 分钟演示路径。按顺序走，每步标注展示内容、讲解话术与异常兜底。

前置准备：后端已启动、前端 `npm run dev`、已登录普通用户账号、管理员口令备用。

---

## 1. 首页与项目定位（约 30 秒）

**展示：** `/` 首页。

**话术：** 这是校园猫猫协作平台，核心闭环是「识别 → 记录偶遇 → 贡献画像 → 社区回流」。首页展示附近猫猫与快速入口。

**兜底：** 首页空数据时展示引导卡片，不影响后续流程。

## 2. 识猫 / 识别流程（约 1 分钟）

**展示：** 进入 `/scan`，上传或拍摄一张猫猫照片。

**话术：** 上传后调用 AI 识别，返回 confirmed / uncertain / unknown / unavailable 四种状态。后端已有 DINOv2-style 本地模型与 reference embedding 链路，模型不可用时会安全降级。

**兜底：** 若识别服务不可用，页面展示「识别服务暂时不可用」并保留手动提交入口，不卡死。

## 3. 确认或处理 uncertain / unknown（约 45 秒）

**展示：**
- confirmed：直接跳转偶遇确认。
- uncertain：展示 Top3 候选，手动选择一只。
- unknown：提示提交到「新猫线索」。

**话术：** 不确定结果由用户兜底确认，保证数据可信；未知猫进入线索队列等管理员审核。

**兜底：** 候选列表为空时直接走 unknown 流程。

## 4. 猫猫档案与照片墙（约 45 秒）

**展示：** 从识别结果或首页点击进入 `/cats/:catId`，展示档案信息与照片墙。

**话术：** 每只猫有独立档案：性格、绝育、健康记录、偶遇历史。照片墙聚合社区上传。

**兜底：** 无照片时展示占位图，不空白。

## 5. 地图与路线推荐（约 1 分钟）

**展示：** 进入 `/map` 展示热力图；再进入 `/routes` 切换时间段查看推荐路线。

**话术：** 地图基于高德 API 展示猫猫分布；路线推荐基于近 14 天偶遇热度按时间段聚合，可分享并跳转识猫打卡。

**兜底：** 高德 key 未配置时地图区域展示提示；路线无数据时展示「先记录偶遇」空状态。

## 6. 社区与分享闭环（约 45 秒）

**展示：** 进入 `/community` 浏览帖子；点击一条帖子查看详情与评论；从偶遇确认页跳转分享页 `/sightings/share/:id`。

**话术：** 社区支持发帖、点赞、评论、举报。偶遇分享页可外链传播，带回流入口。

**兜底：** 分享页公开可访问，无需登录；帖子加载失败展示错误提示。

## 7. 个人贡献画像（约 30 秒）

**展示：** 进入 `/profile`，展示贡献分、分类 breakdown、徽章、关注列表、周报。

**话术：** 用户每次识别 / 偶遇 / 线索都累计贡献分，分类型排行榜在 `/league`。

**兜底：** 新用户贡献为 0 时展示引导去识猫。

## 8. 管理端总览与审核流（约 1 分钟）

**展示：** 进入 `/admin`，输入管理员口令登录，展示总览 tab；切换到猫档案 / 举报 / 偶遇 tab 演示审核。

**话术：** 管理员总览看板展示待审线索、举报、偶遇与热区。审核动作写入 AuditLog，可追溯。

**兜底：** 口令错误返回 401 不泄漏是否存在管理员；总览数据为空时各卡片展示 0。

## 时间分配

| 步骤 | 时长 |
|------|------|
| 1 首页 | 30s |
| 2 识别 | 60s |
| 3 确认 | 45s |
| 4 档案 | 45s |
| 5 地图路线 | 60s |
| 6 社区分享 | 45s |
| 7 贡献画像 | 30s |
| 8 管理端 | 60s |

合计约 6 分钟，预留 2 分钟答疑。
